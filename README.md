# simple_flask_api
Simple Web API using Flask


## Steps to run app

 ``` virtualenv --python=python3 pyenv3 ```

 ``` source pyenv3.7/bin/activate ```

 ``` pip install flask ```
 
 ``` git clone https://github.com/AppayRocks/simple_flask_api.git ```

 ``` cd simple_flask_api ```

 ``` python app.py ```
